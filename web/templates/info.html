-How does this works?-

The bot operates on a simple principle: it buys when the RSI indicator reaches a certain level and sets up buy/sell grid lines. If the price crosses the 'buy line' from below (indicating an upward trend after a decline), the bot averages the position and adjusts the grid lines based on the new average order price. If the price crosses the 'sell line' from above (signaling a downward trend after a rise), the bot sells and the cycle restarts.

-Settings-

The bot offers extensive customization options for flexible and precise trading strategy configuration.

Warning! if u want to change your bots settings turn it off firstly!!!

I. Bot Configuration Settings:

Pair: Trading pair for the bot (e.g., BTCUSDT, ETHUSDT, SOLUSDT).

Interval: Candlestick timeframe used for RSI calculation (options: 1, 3, 5, 10, 15, 30, 60, 120, 240, 360, 720).

OrderPrice: USDT amount per buy/averaging order (min: 5.5).

RSI: Indicator value triggering grid initialization (recommended: 35-42).

BuyStep/SellStep: Grid line spacing adjustments after buy/averaging events.

TelegramNotifies: Enable/disable trade notifications (requires BOT_TOKEN and CHAT_ID if enabled).

II. Environment Variables (for Bybit account integration):

ACCOUNT_TYPE: Bybit account type (default: UNIFIED).

API_KEY: Your Bybit API key (see setup guide below).

API_KEY_SECRET: Your Bybit API secret key (see setup guide below).

BOT_TOKEN: Telegram bot token for notifications (see setup guide below).

CHAT_ID: Your Telegram user ID (see setup guide below).
